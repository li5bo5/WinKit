
# WinKit - 统一桌面助手开发计划

要求极致的轻量、简洁

我准备做个windows11的系统功能增强工具的集合。
https://github.com/li5bo5/PinToDesk
https://github.com/iKineticate/BlueGauge
https://github.com/StraightOcean/MiniPaste
https://github.com/liaozhu913/MouseGestureStudio
将以上合并成一个项目WinKit。|
分步实现。
1.先实现PinToDesk和MiniPaste两个项目的合并
2.完成以后再考虑BlueGauge和MouseGestureStudio的合并，

实现1：

1.以pintidesk为基础，保留pintodesk的全部功能
2.对MiniPaste进行小幅度调整。
3. MiniPaste全局热键锁定为win+v，并删除其他热键设置。
4. PinToDesk和MiniPaste的托盘合并成一个
5.托盘图标右键显示
   - MiniPaste菜单
   - PinToDesk菜单
6. 对minipaste的界面进行美化，使用pintodesk的风格

实现2：
需要BlueGauge显示我的蓝牙设备的电量，当设备接入时，在托盘显示电量图标，断开时不显示电量图标，低于10%电量提示​‌
需要鼠标右键手势，切换任务、关闭程序、切换桌面、执行快捷键、执行程序或脚本
